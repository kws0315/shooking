## 📝 고객 요구사항 정리
1. 모바일 환경에서의 사용성 중심
2. 신발의 사진이 잘 보이도록 작업
3. 장바구니에 상품을 추가할 때, 실시간으로 상품 개수가 표시
4. 작업 결과물을 확인할 수 있는 테스트 URL 제공

## 📋 기능 목록
- 상품 리스트
- 상품을 2개의 컬럼으로 보여준다.
  - `<ProductList />`
- 이미지가 포함된 상품 카드
- 상품 카드에서 담기 버튼과 담김 상태 변화
- 담기 버튼을 누른 후, 헤더에 담긴 상품의 갯수 변화
