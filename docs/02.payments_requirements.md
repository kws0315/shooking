# 결제 모듈 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
패션 쇼핑몰 경제 모듈

### 사용하게 될 기술: 
프론트엔드: Figma, React, BootStrap
백엔드: Django

### 문서 작성자: 
너구리37

## 📝 고객 요구사항 정리
온라인 결제기능 탑재해야한다.
사용자는 신용카드를 등록할 수 있다.
결제모듈의 보안을 철저히 대비해야한다.
사용자의 카드 뒷면 8자리와 보안코드의 정보는 숨김처리해야한다.
사용자는 여러 카드를 등록할 수 있다.

## 📋 기능 목록
- 카드 번호 input
  - `<CardNumberInput />`
- 입력한 카드 이름
  - `<CardName />`
- 카드번호 보안처리
  - `<CardNumberMasking />`
